Para mag run boi 
1. python -m pip install -r requirements.txt
2. uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload  (Eto yung pag rurun mo sya)

Note: Try mo lang muna display sa frontend mo yung camera boi yung live stream naka ready na lahat yan yung ibang services ako na bahala display mo lang muna sya.

Example use to front end 

1. const ws = new WebSocket("ws://BACKEND_IP:8000/ws/camera");

2. ws.onmessage = (event) => {
const base64Frame = event.data;

  setImageUri(`data:image/jpeg;base64,${base64Frame}`);
};

3.<Image
  source={{ uri: imageUri }}
  style={{ width: 320, height: 240 }}
/>

Example use to front end (Notification)

1. const WS_URL_NOTIF = 'ws://BACKEND_IP:8000/ws/notify'; // Replace with your actual backend IP
2. const ws = new WebSocket(WS_URL_NOTIF);
3.ws.onopen = () => {
      console.log('WebSocket connected for notifications');
    };  //Pang check if connected or not

4.ws.onmessage = (event) => {
      // Update notification with the received message
      const notifMessage = event.data;  // Assuming event.data is a plain text message
      setNotification(notifMessage);
    };
