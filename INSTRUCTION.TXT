Prerequisites

Python 3.8 or higher
Webcam/Camera connected to your computer
Both backend and frontend devices on the same WiFi network


Installation
1. Install Dependencies
bashpython -m pip install -r requirements.txt
2. Run the Server
bashuvicorn app.main:app --host 0.0.0.0 --port 8000 --reload

Camera not showing

Check if webcam is connected
Verify BACKEND_IP is correct
Check terminal for errors

Notifications not appearing

WebSocket must be connected
Check browser/app console for errors
Verify /ws/notify endpoint is accessible

"Connection refused"

Ensure server is running
Both devices must be on same WiFi
Check firewall isn't blocking port 8000

Wrong IP Address

Stop server (CTRL+C)
Restart: uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
Copy new IP from terminal


Notes

Dynamic IP Detection: Server automatically detects its IP on startup
Auto-Discovery: Frontend can use /api/server-info to get WebSocket URLs
Notifications: Server sends 3 heartbeat messages, then "Hello", then periodic test messages
Camera Stream: Approximately 30 FPS at 640x480 resolution with 60% JPEG quality
